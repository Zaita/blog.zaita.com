<h1 id="azure-service-principals-for-terraform">Azure Service Principals for Terraform</h1>
<h2 id="overview">Overview</h2>
<p>I want to use Azure DevOps to run Terraform scripts that provision objects in my Azure Tenancy.</p>
<p>Create Service Principal via Azure CLI<br>
Create Resource Groups for Application/Project<br>
Assign IAM on Resource Group for Service Principal<br>
Do not give Service Principal Contributor access to the subscription</p>
